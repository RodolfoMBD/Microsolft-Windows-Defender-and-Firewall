## Description

This guided lab is about reviewing and updating threat definitions on Windows Defender, running the Defender Antivirus quick scan, and configuring Firewall Rules with and without Advanced Security.


## Languages and Technologies Used

- <b>Windows Defender</b>
- <b>Windows Firewall</b>
- <b>Windows Security Center</b>
- <b>Powershell</b>

## Environments Used 

- <b>Windows 11</b> (23H2)

This project helped improve my skills with Microsoft Windows Defender and Firewalls to enhance the security posture of Windows-based systems. The research and process made me further understand the importance of regular updates and scans by utilizing advanced features to create granular rules that can strengthen threat detection, network protection, content filtering, and an overall improvement of the endpoint security posture.
